# reactnative-firebase-example

A simple memo App structured by Expo and Firebase.  
Requires sign up and login, Provides CRUD of a memo has title and body.

## Dev

Install dependencies
```shell
$ yarn
```

Start Expo
```shell
$ yarn start
```
